# ModelParser

ModelParser is tiny library for android JSON parsing. Not use any 3rd party library, it has zero dependencies.
  
## Requirements:
Android api 4.3+:

Add to dependencies:
```
compile 'com.github.gerp83:ModelParser:1.00'
```
